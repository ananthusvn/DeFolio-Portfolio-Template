# DeFolio Portfolio Template

DeFolio is a modern, responsive, and feature-rich portfolio website template designed for web developers, UI/UX designers, and creative professionals. It provides a polished and professional way to showcase your skills, projects, and experience.

**[Live Demo](https://defolio.inbondsolutions.com/)**

## Features

- **Responsive Design**: Fully responsive layout that works perfectly on desktop, tablet, and mobile devices.
- **Dark/Light Mode**: Built-in theme toggler with persistent preference validation using LocalStorage.
- **Interactive UI**:
  - Typing text animation in the hero section.
  - Smooth scrolling navigation.
  - Parallax effects.
  - Intersection Observer animations on scroll.
- **Sections**:
  - Hero with call-to-action.
  - About Me with stats counter.
  - Skills & Technologies grid.
  - Featured Projects carousel with 3D tilt effects.
  - Pricing Plans.
  - Client Testimonials carousel.
  - FAQ accordion.
  - Contact form.
- **Modern Tech Stack**: Built with the latest web standards.

## Technologies Used

- **HTML5**: Semantic and SEO-friendly markup.
- **CSS3**: Custom styles with variables for theming.
- **Bootstrap 5.3**: Robust grid system and components.
- **JavaScript (jQuery 3.7)**: Dynamic interactions and animations.
- **Font Awesome 6.4**: Vector icons.
- **Google Fonts**: Roboto typeface.

## Setup & Installation

This is a static website template, so no complex build process is required.

1. **Clone or Download** the repository.

   ```bash
   git clone https://github.com/ananthusvn/DeFolio-Portfolio-Template.git
   ```

2. **Open the Project**:
   - You can simply open `index.html` in your browser.
   - For the best experience, run it using a local development server like:
     - [Live Server](https://marketplace.visualstudio.com/items?itemName=ritwickdey.LiveServer) (VS Code Extension)
     - `php -S localhost:8000`
     - XAMPP / WAMP / MAMP

## Customization

1. **Personal Info**: Open `index.html` and replace the placeholder content (Name, Title, Descriptions) with your own.
2. **Images**: Replace the images in the `img/` directory with your project screenshots and profile picture.
3. **Links**: Update the navigation links, social media links, and contact information.
4. **Colors**: Modify the CSS variables in `styles.css` to change the color scheme.
   ```css
   :root {
     --primary-color: #your-color;
     --secondary-color: #your-color;
   }
   ```
5. **SEO & Metadata**: Update the `<title>`, `<meta>` description, and Open Graph tags in the `<head>` section of `index.html`.
